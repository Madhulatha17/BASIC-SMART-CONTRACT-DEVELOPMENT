# Smart Contract Overview  

The smart contract performs the following functions:

- Creates a fixed supply of tokens at deployment  
- Stores token balances for each wallet address  
- Enables secure token transfers between users  
- Emits events to track transactions on the blockchain  



# Deployment Steps  

The smart contract is deployed using **Remix IDE** and **MetaMask wallet** on the **Sepolia test network**.

## Steps:

- Open Remix IDE → https://remix.ethereum.org  
- Create `Token.sol` file and paste the smart contract code  
- Compile using Solidity compiler (v0.8.20 or above)  
- Connect MetaMask wallet  
- Select **Sepolia Test Network**  
- Deploy contract with initial token supply  
- Verify deployment and interact with contract functions
