#  Smart Contract Deployment Steps

##  Tools Required
- Remix IDE (https://remix.ethereum.org)
- MetaMask Wallet
- Ethereum Test Network (Sepolia)


##  Steps to Deploy

### 1. Open Remix IDE
Go to https://remix.ethereum.org



### 2. Create File
Create:
Token.sol
Paste smart contract code



### 3. Compile Contract
- Go to Solidity Compiler tab
- Select version 0.8.20
- Click Compile Token.sol



### 4. Connect Wallet
- Open Deploy & Run tab
- Select "Injected Provider - MetaMask"
- Connect to Sepolia Test Network



### 5. Deploy Contract
- Enter initial supply (example: 1000)
- Click Deploy
- Confirm transaction in MetaMask



### 6. Test Transfer Function
- Use transfer function
- Enter recipient address and amount
- Confirm transaction
